# Walmart Sales Analysis using Python & SQL

## 📌 Project Overview

This project analyzes Walmart sales data using **Python** for Exploratory Data Analysis (EDA) and **SQL** for business insights. The objective is to understand sales performance, holiday impact, economic factors, and store-wise trends to support data-driven decision-making.

The project combines data visualization, statistical analysis, and advanced SQL queries to extract meaningful business insights from Walmart's historical sales dataset.

---

# 🎯 Objectives

- Perform data cleaning and preprocessing.
- Explore sales trends across stores.
- Analyze the impact of holidays on sales.
- Study the influence of economic indicators.
- Compare store performances.
- Apply SQL Window Functions for advanced analytics.
- Visualize important business insights.
